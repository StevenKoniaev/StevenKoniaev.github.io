Neural Network From Scratch

Studying neural networks myself, I created from scratch a convolutional neural network using the MNIST data set. The neural network takes a 28x28 pixel image (784 inputs). The pixel values are then normalized and passed through the network (using sigmoid function for a pass) and then outputted with a Softmax function in order to make the prediction. The values are then adjusted through the optimization of a cost function, using the gradient of the cost function. In this case the Cross Entropy loss function.

In the application you can load in an existing set of weights for the network or train it yourself. There is also an application to see how well the network recognizes your hand writing.

The application is avaliable under releases in the Github respository.




Array Visualizer

Using the React Framework I created an array visualiser website. The program can create a randomly sorted array, the values are represented by the bar's heights. Then a sorting algorthim can be selected and an animation will generate.

https://stevenkoniaev.github.io/ArrayVisualiser/

The website can sort with Merge Sort, Quick Sort, Heap Sort, Insertion Sort, Selecton Sort, and Bubble Sort. The animation occurs when two values are swapped so algorthims that have many comparisons but little swaps appear to be very fast such a Selection Sort which linearly finds the smallest element in an array and brings it to the front. The colours of the bars also change to red as they are being compared and then to green to show that they have been adjusted.
